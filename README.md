# Sistema de Gestión de Eventos y Asistencias

Un sistema para gestionar eventos y registrar asistencias con generación de códigos QR y envío de emails de confirmación.

## Características

- Gestión completa de eventos (CRUD)
- Registro de asistencias con validación de email
- Generación automática de códigos QR para cada asistente
- Envío de emails de confirmación con SendGrid
- Interfaz responsiva y amigable

## Requisitos

- PHP 7.4 o superior
- MySQL 5.7 o superior
- Composer
- Extensiones PHP: PDO, GD, mbstring
